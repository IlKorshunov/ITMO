# Умножение матриц [![CI/CD](../../actions/workflows/classroom.yml/badge.svg?branch=main&event=workflow_dispatch)](../../actions/workflows/classroom.yml)

Чтобы GitHub Workflow отработал верно, файл с [функцией `main`](https://en.cppreference.com/w/c/language/main_function) должен располагаться в корне репозитория.

При сборке компилируются `'*.cpp'`. Допускается вынесение кернел-кода в отдельные файлы, расширение таких файлов – `.cl`.
